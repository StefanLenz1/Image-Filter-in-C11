# Image-Filter-in-C11

A program that takes .bmp files and filters them

## Usage

```
./imagefilter -(flag) infile.bmp outfile.bmp
```
## Possible flags

- "-b": blur
- "-e": marks edges
- "-g": makes it black and white
- "-r": reflects them
- "-m": motion blur
